# Personal Portfolio Website
A modern and responsive personal portfolio website designed to showcase professional background, skills, and projects in a visually appealing and interactive layout.

## Description
The **Personal Portfolio Website** serves as a digital representation of professional identity. It highlights essential details such as personal bio, CV, skills, and completed projects.  
The site features dynamic background images, project carousels, and clean card-based sections, providing visitors with an engaging and smooth browsing experience.  

This project is ideal for showcasing technical skills, design sense, and development experience — whether for personal branding, job applications, or showcasing academic work.

## Getting Started

# Dependencies
Before running the project, ensure you have the following:
- A modern web browser (e.g., Google Chrome, Microsoft Edge, Firefox)
- Operating System: Windows 11, 10 or later / macOS / Linux
- (Optional) A text editor such as VS Code for editing source files

# Installing
1. **Download or Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git

# Open the Project Folder

Navigate to the folder in your file explorer or using the terminal.

# Modify the Content

Replace profile information, CV details, and project images in the respective folders:

index.html – Update text content (bio, CV info, project titles)

style.css – Customize colors, font sizes, or spacing

images/ – Add your own background and project images

# Executing Program

Open the project folder.

Double-click on index.html to open it in your default browser.

The portfolio should load automatically, displaying:

A dynamic background image slider.

Profile section at the center.

Cards for CV and About.

Project images that slide within small display windows.

# Example: Run locally using a lightweight server (optional)
python -m http.server 8080


Then open your browser and go to:

http://localhost:8080

# Help

If images are not displaying correctly:

Check that all image paths in index.html point to the correct folder.

Ensure image file names are correct and have supported formats (e.g., .jpg, .png).

If the layout appears off:

Try adjusting CSS properties like background-size: cover; or container widths.

For JavaScript-related issues:

# Open browser console to check errors
Ctrl + Shift + J
